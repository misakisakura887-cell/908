# Mirror-AI 产品需求文档 (PRD v1)

> 最后更新：2025-02-11
> 版本：v1.0 - 初版

---

## 一、产品定位

**人人可用的 AI 投资平台**

通过 AI 量化交易策略，让普通用户也能参与虚拟资产和 RWA 投资。

---

## 二、核心基础设施

### Hyperliquid (HL)
- **类型**：去中心化交易所 (DEX)
- **关键特性**：无需 KYC
- **作用**：作为底层交易执行层，为用户投资虚拟资产提供通道

---

## 三、核心功能模块

### 1. 钱包连接
- 支持主流钱包：MetaMask、Uniswap Wallet 等
- 用户通过钱包登录/授权

### 2. 策略展示

#### 首页 - 策略排行榜
- 最近表现最佳策略
- 7日/30日 ROI 排名
- 关键指标展示（收益率、回撤、夏普比率等）

#### 策略广场
- 按**交易对**分类（BTC/USDC, ETH/USDC...）
- 按**策略类型**分类（趋势跟踪、网格、套利...）
- 策略详情页：历史表现、风险指标、创建者信息

### 3. 策略创建（成长路径）
- 平台自建 AI 量化策略
- 开放给优秀策略师创建和发布
- 其他用户可跟单第三方策略

### 4. 投资/跟单流程
1. 用户选择策略
2. 唤起钱包插件
3. 输入投资金额 → Swap 到平台托管地址
4. AI 量化机器人执行交易
5. 收益实时结算

### 5. 用户个人中心
- 已跟单的策略列表
- 每个策略投入金额
- 当前盈亏（金额 + 百分比）
- 累计交易手续费
- 提现/追加投资操作

---

## 四、商业模式

- **平台手续费**：从用户交易中抽取微量手续费
- **策略分成**（可选）：第三方策略师的收益分成

---

## 五、技术需求

| 模块 | 说明 |
|------|------|
| Hyperliquid API | 交易执行、行情获取、账户管理 |
| AI API | 策略信号生成、风控决策 |
| 钱包集成 | Web3 钱包连接（ethers.js / wagmi） |
| 用户后台 | 账户管理、投资记录、收益统计 |
| 安全防护 | 防 MEV 攻击、防网络劫持、智能合约审计 |
