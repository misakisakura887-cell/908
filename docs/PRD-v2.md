# Mirror-AI 产品需求文档 (PRD v2)

> 最后更新：2025-02-11
> 版本：v2.0 - 商业模式 & 策略更新

---

## 一、产品定位

**人人可用的 AI 投资平台**

核心理念：**1 美金起投**，极低门槛让普通人也能参与专业级投资策略。

---

## 二、核心基础设施

### Hyperliquid (HL)
- **类型**：去中心化交易所 (DEX)
- **关键特性**：无需 KYC
- **支持资产**：
  - 加密货币（BTC、ETH、Meme 币等）
  - **美股**（Traditional Finance 板块）
- **作用**：底层交易执行层

---

## 三、接入与跟单模式

### 用户接入
- **最低门槛**：1 美金
- **钱包支持**：任意钱包类型
  - 软件钱包（MetaMask、Trust Wallet 等）
  - 硬件钱包（Ledger、Trezor 等）
  - 热钱包 / 冷钱包
- **流程**：连接钱包 → 对接 API → 选择策略 → 开始跟单

### 跟单机制
- 用户选择策略后，系统自动按比例分配资金
- 1 美金也能分成十几份投入不同标的
- 实时显示仓位和价格变化

---

## 四、策略体系

### 策略来源演进

| 阶段 | 来源 | 说明 |
|------|------|------|
| 初期 | 平台自建 | TAKI 团队编写并上线策略 |
| 中期 | 社区创建 | 普通用户、大神、KOL 创建策略 |
| 后期 | 生态繁荣 | 策略市场，创作者经济 |

### 激励机制
- 策略创建者获得跟单手续费分成
- 交易产生时自动结算

### 初期策略规划（目标：几十个主流策略）

#### 1. AI 产业链概念股
- **Mag 7 七巨头**：AAPL、MSFT、GOOGL、AMZN、META、NVDA、TSLA
- **芯片板块**：NVDA、AMD、INTC、AVGO、QCOM
- **存储板块**：MU、WDC、STX
- **AI 应用**：PLTR、AI、PATH

#### 2. Meme 币策略
- 加密板块复苏行情
- 热门 Meme：PIPPIN、DOGE、SHIB、PEPE、WIF 等
- 趋势跟踪 + 动量策略

#### 3. 大师级策略

**巴菲特策略**
- 跟踪伯克希尔 13F 持仓
- 按比例分配（即使 1 美金也分成十几份）
- 季度再平衡

**国会山女股神策略**
- 跟踪 Nancy Pelosi 等议员公开持仓
- 数据来源：国会交易披露

#### 4. 数据爬取策略

**Rockflow 跟单**
- 爬取平台带单大神策略
- 复制其交易信号

**AutoPilot 跟单**
- 调用 AutoPilot API
- 获取主流大神持仓数据

## 五、商业模式

### 收费原则：**用户友好，无感付费**

| 收费项 | 金额 | 说明 |
|--------|------|------|
| License Fee | **0** | 不收取软件授权费 |
| 月费/年费 | **0** | 不收取订阅费 |
| 交易手续费 | 正常费率 + 小额加成 | 仅在交易时收取 |

### 示例
- 100 美金交易
- 原手续费：~3 美金
- Mirror-AI 收取：~5 美金（+2 美金用于 AI/Token 成本）
- 用户感知：几乎无感

### 费用用途
- AI 智能体调用成本
- Token 消耗
- 平台运营

| 模块 | 说明 |
|------|------|

## 七、用户价值主张

1. **极低门槛**：1 美金起投
2. **专业策略**：大师级、AI 量化策略触手可及
3. **零固定成本**：不用不付费
4. **实时透明**：随时查看仓位和收益
5. **去中心化**：资产自持，钱包自控
